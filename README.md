# Harness Engineering 插件仓库

本仓库是面向下游 AI Coding 项目的插件与 scaffold 仓库，包含 Harness Engineering 相关的安装指南、插件约定和可复用文档。

## 执行纪律

- 先澄清，再动手
- 只改必须改的地方
- 用可验证结果收尾

## 目录结构

| 目录 | 说明 |
|------|------|
| `harness-for-enterprise/` | Harness Enterprise 版本安装指南 |
| `openspec-opc/` | OpenSpec OPC 插件安装与配置指南 |
| `docs/gstack/` | gstack 使用文档与速查资料 |
| `docs/forgeflow/` | ForgeFlow 插件设计记录与工作流草稿 |

## 快速开始

请根据您的需求进入对应目录查看详细的插件说明或安装说明：
- **gstack**: 查看 [`docs/gstack/01-quickstart.md`](docs/gstack/01-quickstart.md)
- **OpenSpec OPC**: 查看 [`openspec-opc/README.md`](openspec-opc/README.md)
- **ForgeFlow**: 查看 [`docs/forgeflow/README.md`](docs/forgeflow/README.md)
